# 🌤️ SkyFetch - Weather Dashboard
A beautiful, interactive weather dashboard that provides real-time weather data and 5-day forecasts for any city.

## ✨ Features
- 🔍 Search weather for any city worldwide
- 🌡️ Current temperature, weather conditions, and icon
- 📊 5-day weather forecast
- 💾 Recent searches saved using localStorage (max 5)
- 🔄 Auto-load last searched city
- 🧹 Clear search history
- 📱 Fully responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Axios
- OpenWeatherMap API
- localStorage

## 🚀 How to Run
1. Open `index.html` in browser
2. Search any city name
